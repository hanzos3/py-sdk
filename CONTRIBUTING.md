# Contributors Guide
``py-sdk`` welcomes your contribution. Below steps can be followed to create a pull request.

* Fork this py-sdk repository into your account.
* Create a feature branch in your fork (`$ git checkout -b my-new-feature`).
* Hack, hack, hack...
* Run checks. (`$ make check`).
* Commit your changes (`$ git commit -am 'Add some feature'`).
* Push the feature branch into your fork (`$ git push origin -u my-new-feature`).
* Create new pull request to `main` branch.
