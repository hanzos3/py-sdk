### Setup py-sdk Github Repository
Fork [py-sdk upstream](https://github.com/hanzos3/py-sdk/fork) source repository.

```sh
$ git clone https://github.com/$USER_ID/py-sdk
$ cd py-sdk
$ python setup.py install
...
```

### Developer Guide

``py-sdk`` welcomes your contributions:

* Fork the project
* Create a feature branch (git checkout -b my-new-feature)
* Commit your changes (git commit -am 'Add some feature')
* Push to remote branch (git push origin my-new-feature)
* Create a Pull Request
